# In order to run a project

Go to
```
..\Source\UI\ClientApp
```
Run
```
npm install
```

Then Open VS solution and build project.

Run UI project.

Then open another Visual Studio and run tests.
You can run tests from the same solution, but this way it would be easier for you.
